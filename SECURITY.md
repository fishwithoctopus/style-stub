# Security Policy

Style Stub 是本地优先的桌面工具，但可通过本机网关调用用户自行配置的第三方 AI API。

## 密钥与个人数据

- 桌面版 API Key 使用 Electron `safeStorage` 调用 Windows 用户级加密能力，密文保存在当前用户的应用数据目录。
- 网页开发版的 Key 只存在于本地网关进程内存，停止进程后清除。
- 前端不会读取已保存 Key 的原文；Key 不应进入 localStorage、仓库、日志或截图。
- 上传例图会先在本机缩放，再发送给用户选择的模型提供商。原始图片不写入仓库；个人馆藏只保存本机压缩封面。

使用第三方模型意味着相关请求会受该提供商的隐私政策和数据处理条款约束。请勿上传没有权利处理的素材或包含敏感个人信息的截图。

## 报告问题

如果发现脚本注入、密钥泄露、私人馆藏泄露、不安全导入或网关绕过问题，请不要在公开 Issue 中粘贴真实 Key、个人数据或可直接利用的敏感细节。仓库公开后请使用 GitHub 的私密安全报告功能；若该功能暂不可用，请先只提交不含利用细节的联系请求。

## 提交前检查

贡献者不得提交 API Key、访问令牌、登录信息、私人文件路径、用户数据目录、个人馆藏或未获许可的素材。
