## 正文

最近一直在想一件事：

我们收藏了很多好看的 UI，但真正要交给 AI 做页面时，常常只剩下“高级一点”“像苹果一点”“有纸感”这种很难执行的话。

所以我做了 **Style Stub / 风格票根**。

它是一款可以钉在 Windows 桌面侧边的小工具，也是一本文字、例图和 Prompt 都能一起查的视觉风格图鉴。

目前先放了 35 张票根，包括包豪斯、瑞士国际主义、现代极简、8-bit 像素、Y2K、拟物主义、iOS 7、Liquid Glass，以及一些带日期的当代产品案例。

每张票根不只写“这是什么”，还会给：

- 原创 CSS 视觉小样
- 视觉机制和常见组件
- 适合用在哪里、最容易怎么用错
- 一份可以直接复制给 Codex、Claude Code、v0 或其他工具的中英文 UI Prompt

我还加了一个 **Style Lab**。可以把 2–4 种风格放进去，自由拉每一种的份量，后台再换算成百分比，组合成一份新的风格配方。

“我的风格”则可以保存 Lab 里诞生的配方，也可以用 1–4 张自己的例图提取视觉规则。这里最想解决的是“风格迁移”和“内容复刻”的区别：AI 会先看图提取证据，再做一次不带图片的文本复审，主动排除原页面的产品名、Tab、专属图标和信息架构。

它现在已经是一个真正的 Windows 桌面小工具：能置顶、贴边、记住位置，也能藏进系统托盘。个人馆藏只保存在本机；API Key 由 Windows 用户级加密保存，不会进 GitHub。

下一步准备把它以 MIT License 开源，也欢迎大家以后一起补票根、修定义和测试 Prompt。
