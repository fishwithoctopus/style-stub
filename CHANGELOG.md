# Changelog

All notable changes to Style Stub are documented here.

## 0.5.1 — Unreleased

- Changed the license for the current source tree and future versions to PolyForm Noncommercial 1.0.0.
- Clarified that commercial use requires separate written permission from the copyright holder.
- Dedicated the built-in `promptZh` and `promptEn` texts under CC0 1.0 so prompts remain freely usable in commercial and noncommercial projects.
- Clarified that Style Stub claims no ownership over user-entered or user-generated prompt output and imposes no additional usage restriction on it.
- Preserved the MIT License for the already-published `v0.5.0` tag and Windows release; the change is not retroactive.

## 0.5.0 — 2026-07-20

- Added the Windows desktop app with transparent ticket-shaped window, tray menu, always-on-top, edge snapping and position memory.
- Added Windows user-level encrypted API Key persistence.
- Expanded the public catalog to 35 style tickets with original CSS specimens and full bilingual UI prompts.
- Added fuzzy search, editable categories and drag-to-reorder catalog tickets.
- Added a four-style Lab with freely adjustable strengths and normalized weights.
- Added personal-style covers, detail pages, editing, deletion and image-based style extraction.
- Changed the default image workflow to Qwen visual extraction followed by a text-only Qwen review that removes source-product content.
- Added the Style Stub application and tray icons.
- Adopted the MIT License.

## 0.1.0 — 2026-07-20

- Created the first ticket-style visual dictionary and local Lab prototype.
